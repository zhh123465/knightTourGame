# 马踏棋盘游戏 - 命令行版本

## 项目简介

这是一个基于Java实现的马踏棋盘（Knight's Tour）游戏命令行版本。该程序使用非递归回溯算法求解骑士遍历8×8棋盘的问题。

## 功能特性

- ✓ 非递归回溯算法实现
- ✓ 支持指定起始位置求解
- ✓ 支持随机起始位置求解
- ✓ 支持批量求解所有起始位置
- ✓ 显示解矩阵和统计信息
- ✓ 完整的错误处理和输入验证

## 技术特点

- **数据结构**: 使用栈（Stack）管理移动路径
- **算法**: 非递归回溯 + Warnsdorff启发式优化
- **架构**: Model层清晰分离（Position, Cell, Board, Move, Solution）
- **代码规范**: 遵循Java命名规范和Google Java Style Guide

## 编译和运行

### 系统要求

- Java 8 或更高版本
- Maven 3.6 或更高版本（可选）

### 编译项目

```bash
cd knight-tour-cli
javac -d target/classes src/main/java/com/knighttour/**/*.java
```

### 运行程序

```bash
java -cp target/classes com.knighttour.cli.Main
```

## 使用说明

### 主菜单选项

1. **从指定位置开始求解**
   - 输入起始位置的行和列坐标（0-7）
   - 程序将求解并显示解矩阵

2. **从随机位置开始求解**
   - 程序随机选择一个起始位置
   - 自动求解并显示结果

3. **求解所有起始位置**
   - 批量求解所有64个起始位置
   - 显示每个位置的求解结果和统计信息

4. **退出**
   - 退出程序

### 输入格式

- 行坐标: 0-7 的整数
- 列坐标: 0-7 的整数
- 起始位置示例: (0, 0) 表示左上角

- **验证结果**: 显示解是否有效

## 算法说明

### 非递归回溯算法

本程序使用非递归回溯算法，避免了递归调用栈溢出的问题。算法核心数据结构包括：

1. **MoveStackEntry**: 栈条目，包含位置、序号和未尝试的移动列表
2. **MoveGenerator**: 移动生成器，计算合法移动
3. **KnightTourSolver**: 求解器主类，实现算法逻辑

### Warnsdorff启发式

为了提高求解效率，程序使用Warnsdorff启发式算法：
- 优先选择可行移动较少的位置
- 这可以减少回溯次数，提高求解速度

## 核心类说明

### Position
- 表示棋盘上的位置坐标
- 提供坐标验证和距离计算功能

### Cell
- 表示棋盘上的方格
- 记录访问状态和访问序号

### Board
- 棋盘模型，维护8×8网格状态
- 提供访问控制和状态管理

### Move
- 表示单次移动操作
- 包含起始位置、目标位置和未尝试移动列表

### Solution
- 存储完整的解矩阵
- 提供解验证和格式化输出

### MoveGenerator
- 计算骑士的合法移动
- 实现Warnsdorff启发式优化

### KnightTourSolver
- 核心求解器
- 实现非递归回溯算法
